\begin{abstract}
    As artificial intelligence and complex classifiers become increasingly prevalent in sensitive 
    domains, the ability to understand and trust the decisions of these black-box models is essential. 
    This document provides a comprehensive overview of current explainable AI (XAI) methods designed 
    to interpret classifier outputs. The analysis categorizes these techniques into three primary 
    families: logic-based abductive methods, game theory-based Shapley value approaches, and 
    perturbation-based model-agnostic systems.
    First, logic-based methods attempt to formalize the classifier's reasoning to offer formal 
    guarantees on the quality and rigor of the produced explanations. Second, Shapley value 
    methods evaluate feature importance by assigning a contribution score to each input based on 
    game theory principles. While these scores are easily interpretable for users, computing them 
    can be computationally intractable and heavily dependent on characteristic functions and reference 
    distributions. Finally, perturbation methods observe model behavior by generating perturbed input 
    samples. These methods provide local explanations either through linear approximations or 
    user-friendly IF-THEN rules. Ultimately, the document concludes that the choice of an explanation 
    method depends on the desired balance between formal reliability, computational complexity, and 
    user interpretability.
    \keywords{Explainable AI \and Classifiers \and Black-box models \and Feature Importance}
\end{abstract}
